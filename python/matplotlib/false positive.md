k-means clustering (k 평균 군집)

* 주어진 데이터를 k개의 cluster로 묶어주는 알고리즘
* cluster의 개수, 위치를 결정 
* 각 데이터 표본이 어떤 centroid에 가장 가까운지 판단
* centroid의 위치를 형성된 군집을 기준으로 조정 
* 같은 과정을 개선의 여지가 없을 때까지 반복
