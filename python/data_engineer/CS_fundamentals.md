# CS Fundamentals

## 1. basic terminal usage

* change directory 

  * `cd` 

* listing directory

  * `ls`

* open files

  * `open`

* copy a file to another directory

  * `cp "filename" "newfilename"`

* create a text file

  * `touch`

* create a directory

  * `mkdir`

* remove an empty directory

  * `rmdir`

* execute commands with superuser privileges

  * `sudo`
